# Ignition Anomalies

## Definition
An ignition anomaly occurs when the ignition signal rapidly changes state, remains noisy, or does not align with motion and engine behavior.

## Why It Matters
Ignition anomalies can cause:
- missed trips
- split trips
- inflated idle time
- false engine-on interpretation

## First Checks
1. Inspect installation and ignition wire quality.
2. Review whether the signal is flapping between on and off.
3. Correlate anomalies with error `E104`.
4. Confirm the issue is not simply GPS startup delay.

## Escalation Guidance
If anomalies recur after installation verification, request a wiring inspection and compare against a known-good device profile.
